# Progress Log
Spec: bookmark-page

## Codebase Patterns
- Worker CORS pattern: check Origin against allow-list, return origin or default
- D1 binding via wrangler.toml [[d1_databases]], local dev uses .dev.vars for secrets
- Auth: Bearer token compared to env.API_KEY secret
- Local testing: .dev.vars for API_KEY, wrangler d1 execute --local --file=schema.sql for schema
- Hugo layout pattern: {{ define "main" }} in layouts/<section>/list.html, content/<section>/_index.md
- Theme CSS: .post-item flexbox, .search-article input, .tag-item chips, [data-theme="dark"] for dark mode
- JS pattern: inline <script> at page bottom, API URL from site.Params, IIFE wrapper
